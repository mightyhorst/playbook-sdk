# Complexity Management

There are two primary reasons to control the number of cells displayed at any one time. 

1. The first is performance, drawing more and more cells will reach performance usability limits at some point on any platform. 
2. The second reason is ease of use, a human can only comprehend a certain amount of information. 

All of the concepts associated with grouping, listed above, can be used to reduce the complexity of information on the screen for the user.
